# ShiftMate

ShiftMate is a digital platform designed to simplify **shift management** for businesses with variable staff, such as hotels, restaurants, and operational teams. The app helps **workers** find and claim shifts easily, while **managers** can post shifts and monitor applications in real time.

---

## Features

### For Workers
- View available shifts
- Apply for shifts directly from the app
- Track your upcoming and past shifts

### For Managers
- Post new shifts quickly
- Monitor staff applications
- Organize schedules efficiently

### General
- Dark/light mode support
- Smooth navigation with **Expo Router**
- Type-safe with **TypeScript**
- Styled using **NativeWind** (Tailwind for React Native)

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Carmelo85S/ShiftMateRN.git
cd ShiftMateRN

npm install
# or
yarn install

npx expo start

```

Tech Stack
React Native with Expo
TypeScript for type safety
Expo Router for navigation
NativeWind for styling
Supabase for backend and database management
